## Sobre o jogo

**Cosmic Clicker Chronicles** é um idle clicker com progressão profunda, ambientado no cosmos. Colete Cosmic Dust tocando no cristal galáctico, compre upgrades, e acumule Stellar Energy através do sistema de Prestígio para multiplicar sua produção permanentemente.

**Plataforma:** Android (via APK) · Browser mobile · Landscape exclusivo  
**Engine:** HTML5 puro · Zero dependências externas · Single-file build

---

## Funcionalidades

- **8 upgrades** com custo exponencial e barra de progresso visual
- **6 Stellar Upgrades** permanentes desbloqueados via Prestígio
- **Sistema de Prestígio** com recompensa calculada por raiz quadrada do total de Dust
- **Offline Earnings** — ganhos passivos acumulados em até 4 horas de ausência, com modal animado de "Bem-vindo de volta"
- Aurora boreal animada em CSS puro, estrelas em canvas e glassmorphism
- Auto-save a cada 10 segundos + save ao bloquear a tela

## Como publicar uma nova versão

1. Atualize `index.html` com o novo código do jogo
2. Faça commit com a mensagem `feat: v0.X — [descrição da feature]`
3. O GitHub Pages publica automaticamente em ~2 minutos
4. Para gerar o APK: acesse [Web Into App](https://www.webintoapp.com/), cole a URL `https://coca69643.github.io/cosmic-clicker/` e faça o download do APK
5. Suba o APK gerado como **Release** neste repositório (botão "Create a new release")
